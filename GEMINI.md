# 프로젝트 규칙 (GEMINI.md)

이 프로젝트는 Python 기반의 주식 종목 추천 앱입니다. Discord 봇을 통해 주식 종목을 추천합니다.
작업 시 아래의 기술 스택과 규칙을 **엄격히** 준수하십시오.

## 1. 패키지 관리 (중요!)
- **패키지 매니저:** 무조건 `uv`를 사용합니다. (`pip` 사용 금지)
- **설치 명령어:** `uv add <package_name>`
- **실행 명령어:** `uv run <script_name>` (가상환경 활성화 불필요)
- **초기화:** `uv init` (이미 완료됨)

## 2. 코딩 스타일 (Vibe Coding)
- 코드는 간결하고 직관적으로 작성하십시오.
- 모든 함수에는 Type Hint를 명시하십시오.
- 복잡한 로직보다는 이미 검증된 라이브러리 활용을 우선하십시오.

## 3. SQLite 관련
- DB 파일 경로는 다른 프로젝트에서 갱신함. 따라서 **절대로 이 프로젝트에서 DB를 갱신하면 안됨.**
- discord를 사용함으로 DB 접근시 aiosqlite 를 사용하여 비동기적으로 접근해야함.
